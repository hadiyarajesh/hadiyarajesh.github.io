InstaBot Documentation

Installation

Checkout/clone the instabot project from here.

Usage

Intialize the InstaGramBot class from your main() class as below 
  
val username = "your_username"
val password = "your_password"

val bot = InstagramBot()
Call the prepare() method first and then login() method to login into Instagram with your username and password. (You must call prepare() method first before calling login() method()) 
  
bot.prepare(username, password)
bot.login()
Now, you can call any methods of InstaGramBot class to perform desired action. For exmaple, to get list of 100 followers of specific user and print them, you can call below method on bot object. Here we're using collector lambda function of Flow to asynchronously get user names. 
  
bot.getUserFollowers("user_name", 100, isUsername = true).collect { println(it) }

List of Public methods

This is the list of public methods with their description that bot can use to perform any operation.

---> Prepare()

Prepares the bot object before logging into Instagram.You need to pass your username and password as parameter to this mehtod.

Signature 
  
fun prepare(username: String, password: String)
Parameters
  • username - Your instagram username
  • password - Your instagram password

---> Login()

Login into instagram with given username/password and interactively handle all scenario of Two factor authentication code and/or login confirmation by showing options in prompt. It returns true if login is successful, false if failed.

Signature 
  
fun login(forceLogin: Boolean = false): Boolean
Parameters
  • forceLogin - Caution: Only use if your cookies are invalid. Frequest use of this can lead to ban your account by Instagram. Default is false.

---> Logout()

Logged you out from Instagram

Signature 
  
fun logout(): Boolean

---> getUserInfoByID()

Get info about specified user

Signature 
  
fun getUserInfoByID(userId: String): JSONObject?
Parameters
  • userId - Username of User.
