Get following list of given user

Parameters

amountOfFollowing - Amount of following users to retrieve

isUsername - Whether to retrieve username of following or not (if false, userid will be returned)

isFilterPrivate - Whether to filter private account or not

isFilterVerified - Whether to filter verified account or not

fileNameToWrite - File name to write usernames of following list. This is especially useful when following list is large (> 10K)

isOverwrite - Whether to replace existing usernames file or not

Return
Flow of usernames/userids
